## Hotel Booking Analysis (SQL)

### Overview
This project analyzes hotel booking data using SQL to uncover trends in customer behavior, booking patterns, and cancellations.

### Key Questions
- Which market segment generates the highest revenue?
- How do bookings vary by month?
- What is the ratio of repeat to new customers?

### Skills Demonstrated
- Aggregations and grouping
- Conditional logic (CASE statements)
- Subqueries
- Business-focused SQL analysis
